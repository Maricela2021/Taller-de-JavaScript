Preguntas de Reflexión
1. ¿Cuál es la diferencia entre let y const?
La diferencia principal es que 'let' permite cambiar el valor de una variable, mientras que 'const' no.
- 'let': se usa cuando el valor puede cambiar durante el desarrollo.
- 'const': se usa cuando el valor no debe cambiar después de ser asignado.

2. ¿Qué es el DOM?
El DOM es una representación del HTML como una estructura de objetos que JavaScript puede manipular.

Permite:
- Cambiar texto
- Modificar estilos
- Agregar o eliminar elementos
- Responder a eventos del usuario

3. ¿Qué es un evento?
Un evento es una acción que ocurre en la página web, generalmente causada por el usuario.

Ejemplos de eventos:
- Hacer clic en un botón
- Escribir en un input
- Mover el mouse

JavaScript puede "escuchar" estos eventos y ejecutar código cuando ocurren.

4. ¿Por qué es importante separar HTML, CSS y JS?
Es importante para mantener el código organizado y fácil de mantener.

- HTML: estructura del contenido
- CSS: diseño y estilos
- JavaScript: funcionalidad e interactividad

Separarlos permite:
- Mejor organización
- Reutilizar código
- Facilitar el mantenimiento
- Trabajar en equipo más fácilmente
